🔐 Autenticação & Usuário

|── controllerUsuario

 - controllerCadastrarUsuario (POST /usuarios)
    º recebe no body: nome, email, senha
    º valida campos obrigatórios e formato do email
    º verifica se o email já está cadastrado
    º senha com pelo menos 6 caracteres
    º realiza a criptografia hash da senha com bcrypt
    º cria novo usuário no banco 
    º retorna: 201 Created

- controllerLoginUsuario (POST /login)
    º recebe no body: email, senha
    º verificar email e senha presentes
    º busca usuário pelo email
    º compara senha com hash usando bcrypt
    º gera e retorna token JWT
    º retorna: 200 OK + token

- controllerObterPerfil (GET /perfil)
    º exige token JWT no header Authorization
    º valida e decodifica token
    º busca dados do usuário
    º retorna: 200 OK + dados do usuário

- controllerAtualizarPerfil (PUT /perfil)
    º exige token JWT
    º recebe campos parciais: nome, email, senha
    º se senha enviada: recriptografar
    º se email enviado: verificar se já existe
    º atualiza dados
    º retorna: 204 No Content ou 200 OK

- controllerDeletarPerfil (DELETE /perfil)
    º exige token JWT
    º confirmar existência do usuário
    º deleta usuário e dados vinculados (transações, metas, categorias)
    º retorna: 204 No Content

## 💸 Transações

│── controllerTransacoes

- controllerListarTransacoes (GET /transacoes)
    º exige token JWT
    º aplicar filtros se presentes data, tipo ou categoria
    º retorna: 200 OK + array de transações do usuário

- controllerDetalharTransacao (GET /transacoes/:id)
    º exige token JWT
    º verifica se transação existe e pertence ao usuário
    º retorna: 200 OK + dados

- controllerCriarTransacao (POST /transacoes)
    º exige token JWT
    º recebe: descricao, valor, tipo, categoria_id, data
    º valida tipo (entrada ou despesa), valor positivo
    º salva no banco
    º retorna: 201 Created

- controllerAtualizarTransacao (PUT /transacoes/:id)
    º exige token JWT
    º verifica se transação existe e pertence ao usuário
    º atualiza campos permitidos
    º retorna: 204 No Content

- controllerExcluirTransacao (DELETE /transacoes/:id)
    º exige token JWT
    º verifica se transação existe e pertence ao usuário
    º deleta do banco
    º retorna: 204 No Content

- controllerResumoTransacoes (GET /transacoes/resumo)
    º exige token JWT
    º calcula totais de entrada, saída e saldo
    º retorna: 200 OK + resumo

## 📊 Metas

│──controllerMetas

- controllerCriarMeta (POST /metas)
   º exige token JWT
   º recebe: titulo, valor_atual, valor_alvo, mes, ano, user_id
   º valida valores positivos
   º salva meta no banco
   º retorna: 201 Created

- controllerListarMetas (GET /metas)
   º exige token JWT
   º retorna: 200 OK + array de metas

- controllerAtualizarMeta (PUT /metas/:id)
   º exige token JWT
   º verifica se meta pertence ao usuário
   º atualiza campos válidos
   º retorna: 204 No Content

- controllerExcluirMeta (DELETE /metas/:id)
   º exige token JWT
   º verifica se meta pertence ao usuário
   º deleta do banco
   º retorna: 204 No Content
